# Proyecto de Git y GitHub
Estudiante: Katherine Castagnoli 

## Descripción 
Este priyecto demuestra el uso de un flujo de trabajo profesional con Git. Se implementaron las siguientes etapas: 
- Inicialización de repositorio y primer commit en 'main'.
- Creacion de una rama de funcionalidad 'feature/navbar'.
- Modificación de 'index.html' con una barra de navegación. 
- Fusión de la rama de funcionalidad a la rama principal. 
- Sincronización con el repositorio remoto en GitHub
